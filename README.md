# Wishlist App 🎁

Веб-приложение для управления списком желаний на Flask.

## Функции
- ➕ Добавление желаний
- ✏️ Редактирование
- ❌ Удаление
- 🔄 Переключение статуса
- 🔍 Поиск по названию
- 📊 Сортировка по цене и статусу
- 💰 Подсчёт общей стоимости

## Установка и запуск

```bash
pip install -r requirements.txt
python app.py
